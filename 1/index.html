Github Tutorial

Setting up an account

git config --global user.name "user_name"
git config --global user.email "email_id"

Creating a local repository

git init Repo_Name

Updating repository

git pull https://github.com/user-name/project-name

Adding/updating files

git add .

Committing changes

git commit -m "some_message"

Push to local repository

(Only the first time as I didn't get any need to use it later)
git remote add origin https://github.com/user_name/project-name

Push

git push origin master

Sass Tutorial

Ruby Install

apt-get install rbenv

Sass Install

sudo su -c "gem install sass"
sudo gem install sass

Check Install

sass -v

Compile one time

sass file.scss file.css

Keep compiling

sass --watch file.scss:file.css
